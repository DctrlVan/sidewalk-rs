# sidewalk-rs
 Tool for controlling LED sidewalk 

### Quick Start
Clone repository:
<br>
```git clone https://github.com/paullucas/sidewalk-rs```
<br>
Change current directory:
<br>
```cd sidewalk-rs```
<br>
Install dependencies and build:
<br>
```cargo build```
<br>
Run the app:
<br>
```cargo run```
<br>
